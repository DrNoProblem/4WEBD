# 4WEBD

## Running application

Install docker, you can use docker desktop: https://www.docker.com/products/docker-desktop/   
Open a bach shell in the root project and type : `docker-compose up`    

Now you can connect to the frontend: http://localhost:3000

if the 3000 port does not work, you can open another bach shell in the root project and type : `cd './frontend'` then type : `npm start`

Now you can connect to the frontend: http://localhost:3000
